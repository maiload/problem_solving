## 알고리즘 문제 접근 방법

1. **문제 이해**: 입력/출력 조건, 제약 사항 파악
2. **시간 복잡도 추정**: 입력 크기(N)에 따른 알고리즘 선택 → [시간 복잡도 가이드](./docs/basics/time-complexity.md)
3. **자료구조 선택**: 문제 유형에 맞는 자료구조 결정
4. **알고리즘 설계**: 완전탐색 → 최적화 순으로 접근
5. **구현 및 검증**: 엣지 케이스 확인
